# Assets Directory

This folder contains app assets like images and icons.

## Structure

- `images/` - App images, logos, placeholders
- `icons/` - Custom icons and graphics

## Note

The app uses cached network images for course thumbnails and instructor profiles.
You can add custom logos and branding here.

### Recommended Assets

- `logo.png` - App logo (512x512)
- `splash.png` - Splash screen image
- `placeholder.png` - Default course thumbnail
- `avatar.png` - Default user avatar
